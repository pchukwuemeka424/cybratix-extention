# Icons

Production icons are located here:
- `icon16.png` (16x16 pixels) ✅
- `icon48.png` (48x48 pixels) ✅
- `icon128.png` (128x128 pixels) ✅

## Icon Design

The icons feature the Cybratix branding:
- Dark grey shield with lighter grey border
- White key with circular head
- Yellow eye with dark pupil inside the key head

## Regenerating Icons

If you need to regenerate icons:

1. **Using Node.js** (recommended):
   ```bash
   cd dev-tools
   node create-icons-node.js
   ```

2. **Using Browser**:
   - Open `dev-tools/generate-icons.html` in your browser
   - Click "Generate Icons"
   - Download each icon and save to this folder

All icons are production-ready and optimized for Chrome Web Store.
