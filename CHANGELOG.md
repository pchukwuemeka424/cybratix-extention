# Changelog

All notable changes to Cybratix will be documented in this file.

## [1.0.0] - 2024-11-24

### Added
- Initial production release
- Comprehensive website risk analysis (0-100 score)
- Six risk factor analysis:
  - Domain Age
  - SSL Certificate validation
  - Server Reputation (VirusTotal integration)
  - WHOIS Abnormalities
  - Phishing Pattern Detection
  - Breach History
- Real-time risk badge on all websites
- Extension popup with detailed risk breakdown
- Configurable API keys via config.js
- Local caching (1 hour) to reduce API calls

### Security
- API keys moved to config file (not in version control)
- No hardcoded sensitive data
- All API calls made server-side
- No user data collection

### Production Ready
- All icons generated and optimized
- Development files organized in dev-tools/
- Production build script
- Comprehensive documentation

### Production Release
- Extension ready for production use
- Comprehensive website security analysis
- Professional security assessment tool
