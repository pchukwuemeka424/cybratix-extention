# Disclaimer

## Purpose

This Chrome extension, "Cybratix," provides website security analysis and risk assessment.

## Intended Use

This extension is designed to:

- Analyze website security factors
- Provide risk scores based on multiple security indicators
- Help users make informed decisions about website safety
- Integrate with security APIs for comprehensive analysis

## Features

This extension analyzes:

- Domain age and registration history
- SSL/TLS certificate validation
- Server reputation and malware detection
- WHOIS data analysis
- Phishing pattern detection
- Breach history checking
